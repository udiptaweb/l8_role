<h5>Steps</h5>

    1. Clone the Repo
    
    2. Run composer install
    
    3. add .env and db config
    
    4. generate key
    
    5. install laravel breeze

    6. install laravel-permission package by spatie

    7. Seed RolePermissionSeeder and SuperAdminSeeder
